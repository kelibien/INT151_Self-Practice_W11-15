- รู้วิธีให้หน้าเว็บตอบสนองได้ เช่น เปลี่ยนข้อความ เพิ่ม ลบ element อ่านค่าจาก input ทำให้เริ่มควบคุมหน้าเว็บด้วย JavaScript ได้จริง
- ได้ฝึกให้หน้าเว็บตอบสนองต่อผู้ใช้ เช่น การคลิก พิมพ์ หรือส่งฟอร์ม ช่วยให้เข้าใจการโต้ตอบของเว็บและจัดการพฤติกรรมผู้ใช้ได้ดีขึ้น
- เรียนรู้การดึงข้อมูลจาก API จริง ใช้ async/await และแสดงผลบนหน้าเว็บ ที่จะเอาไปคุยกับ backend
- เข้าใจการจัดการเวลา เช่น นาฬิกา ตัวจับเวลา คำนวณอายุ ทำให้รู้จักการอัปเดต UI แบบ real-time และใช้งาน Date ได้คล่องขึ้น
